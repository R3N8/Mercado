# Mercaro – Online Shop

Mercaro comes from the Latin word mercari, which means “to trade.”
This project is a fully functional online shop built as part of the JavaScript Frameworks course assignment.
The application demonstrates practical use of React and Next.js with TypeScript, API integration, state management, and responsive UI design.

The goal of Mercaro is to create a realistic e-commerce experience while following modern frontend development practices and meeting all specified assignment requirements.

## Features

- Responsive product grid layout
- Login-, Register flow
- Product Details
- Search & Sorting
- Shopping Cart
- Checkout flow
- Contact form
- Toast notification

## Getting Started

Clone the repository, then install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run component tests:

```bash
npm test
```

## Tech Stack

**Framework**: Next.js (React)

**Language**: TypeScript

**Styling**: Tailwind CSS

**State Management**: React Context / Custom Hooks

**Testing**: React Testing Library

**Notifications**: Toast notification library
